Projects

My projects examine how machine learning models behave when text is not simple, balanced, or drawn from the same distribution used during training. The work is mainly grounded in natural language processing, sentiment analysis, and software security.

A common theme across these projects is evidence. I study which part of an input supports a prediction, when a model should avoid making an automatic decision, and how contextual or structural information can make the prediction more faithful to the actual meaning of the text.

Dissertation Projects

ShiftAware-Select
Reliable NLP classification under distribution shift

Text classifiers are often trained and evaluated under a clean experimental setting, but real use is rarely that stable. A model may be asked to classify data from a new domain, a noisier source, or a different writing style. In such cases, the main concern is not only whether the model is accurate, but whether it knows when its own prediction is unreliable.

ShiftAware-Select addresses this setting through calibration, uncertainty estimation, and selective prediction. The framework allows confident cases to be handled automatically while uncertain cases can be deferred for human review. This makes the model more useful in decision settings where a confident wrong answer is more costly than a cautious abstention.

Distribution Shift Selective Prediction Uncertainty Calibration
Status: Dissertation study / manuscript in progress.
CAGLSAv2
Context-aware sentiment analysis with contrastive clauses

Many sentiment mistakes happen when a sentence contains contrast. A user may praise one part of an experience and then reverse the overall judgment with another clause. In these cases, the final sentiment depends on how the clauses interact, not simply on the presence of positive or negative words.

CAGLSAv2 focuses on this clause-level behavior. It models contrastive information so the representation can better reflect how one part of the sentence modifies another. The project contributes to sentiment analysis by treating linguistic structure as part of the prediction process rather than as background text to be compressed into a single sequence representation.

Context-Aware Learning Contrastive Clauses Sentiment Analysis
Status: Completed dissertation study.
Irony-Aware Multitask Sentiment Learning
Sentiment analysis for non-literal social media text

Social media text often carries opinion indirectly. A post may use words that look positive while the actual meaning is negative, especially when irony, sarcasm, or emotional framing is present. This makes sentiment prediction difficult for models that rely heavily on surface wording.

This project connects sentiment learning with related tasks such as irony and emotion prediction. The motivation is simple: when the intended opinion is not literal, auxiliary information can help the model interpret the text more carefully. The result is a framework designed for opinion-rich social media settings where sentiment depends on more than word polarity.

Irony Emotion Multitask Learning Social Media
Status: Completed dissertation study.
DualGraph-RoBERTa for Aspect-Based Sentiment Analysis
Fine-grained opinion modeling with graph structure

Aspect-based sentiment analysis requires a model to connect the correct opinion to the correct target. A sentence may discuss food, service, and price at the same time, with different sentiment toward each aspect. A broad sentence-level representation can mix these opinions and lose the local evidence.

DualGraph-RoBERTa uses transformer representations together with graph-based structure to preserve aspect-specific relationships. The project is aimed at more precise opinion understanding, where the model must identify not only whether the sentence is positive or negative, but which target the opinion belongs to.

ABSA Graph Learning RoBERTa Opinion Structure
Status: Dissertation project / manuscript in progress.
Discourse-Aware Domain Adaptation for Sentiment Analysis
Reliable sentiment transfer across domains

Sentiment models often lose reliability when moved from one domain to another. A model trained on restaurant reviews may not transfer cleanly to product reviews or social media posts because the aspects, writing style, and opinion cues change.

This project studies domain adaptation using discourse-aware and span-level sentiment evidence. The goal is to make transfer more stable by preserving the local reasoning behind each prediction. Rather than only adapting global representations, this work focuses on how sentiment evidence is expressed and organized across domains.

Domain Adaptation Discourse Span Evidence Generalization
Status: Proposed dissertation study.

Security-Focused NLP Project

Hierarchical Severity Classification with CWE Enrichment
Evidence-aware classification for software vulnerability reports

Software vulnerability reports are technical and often lengthy. The key evidence for severity may be concentrated in a small part of the report, while other sentences provide context or background. If a model treats the whole report as a flat block of text, it can miss the evidence that matters most.

This project uses hierarchical modeling to preserve sentence-level information and enriches the representation with CWE-based security knowledge. The work connects NLP with software security by making severity prediction more tied to the technical evidence in the report.

Software Security CWE Severity Classification Hierarchical Evidence
Status: Completed dissertation study.
