Notable Projects

My projects focus on reliable NLP, context-aware sentiment analysis, AI for cybersecurity, speech and audio AI, mobile sensing, and applied machine learning.

Dissertation and NLP Projects

ShiftAware-Select
Reliable NLP classification under distribution shift

Problem: NLP classifiers can make confident mistakes when test data are noisy, shifted, or out of domain.

Outcome: The framework combines calibration, uncertainty estimation, and selective prediction so uncertain cases can be deferred for human review.

Venue / Status: Under review, IEEE ICMLA 2026.
Distribution Shift Calibration Selective Prediction Reliable NLP
CAGLSAv2
Discourse-span attention for length-robust sentiment analysis

Problem: Sentiment can change across clauses, and long text can hide the real opinion evidence.

Outcome: CAGLSAv2 uses discourse-span attention to capture contrast, span-level evidence, and length variation in sentiment analysis.

Venue / Status: Under review, IEEE ICMLA 2026.
Sentiment Analysis Discourse Span Contrastive Clauses Context-Aware Learning
Irony-Aware Twitter Sentiment Analysis
Multi-task learning for non-literal social media text

Problem: Social media posts often use irony, emotion, and indirect meaning, which can mislead sentiment models.

Outcome: The model learns sentiment with irony and emotion signals to improve understanding of non-literal Twitter-style text.

Venue / Status: Submitted, IEEE AIBThings 2026.
Irony Emotion Multi-Task Learning Social Media NLP
DualGraph-RoBERTa
Syntactic and semantic graph fusion for aspect-based sentiment analysis

Problem: A sentence can praise one aspect and criticize another, so the model must connect each opinion to the correct target.

Outcome: DualGraph-RoBERTa combines syntactic and semantic graph evidence with RoBERTa to support aspect-specific sentiment prediction.

Venue / Status: Submitted, IEEE AIBThings 2026.
ABSA Graph Learning RoBERTa Opinion Structure
Discourse-Aware Domain Adaptation

Outcome: The project studies discourse and span-level evidence to support more reliable sentiment transfer across domains.

Venue / Status: In progress.

Security, Speech, and Sensing Projects

Hierarchical Severity Classification with CWE Enrichment
Evidence-aware classification for vulnerability disclosures

Problem: Vulnerability reports are dense, and severity evidence may appear in only a few sentences.

Outcome: The method combines hierarchical sentence modeling with CWE enrichment to support evidence-aware severity classification.

Venue / Status: Submitted, IEEE AIBThings 2026.
Software Security CWE Severity Classification Sentence Evidence
LayerLens
Task-relevant layer disruption analysis for self-supervised speech encoders

Problem: Global audio quality scores do not show which speech encoder layers are disrupted by noise or reverberation.

Outcome: LayerLens compares clean and distorted representations across layers and links disruption to downstream speech tasks.

Venue / Status: Under review, IEEE/ACM TASLP.
Speech Processing Self-Supervised Learning Layer Analysis Model Diagnostics
Vi-Detect
Smartphone-based vibration sensing for component looseness detection

Problem: Fine-grained looseness detection often requires dedicated sensors, which limits scalability.

Outcome: Vi-Detect uses smartphone accelerometers and vibration modeling to support low-cost structural health monitoring.

Venue / Status: ACM SenSys poster published; full article under minor revision at ACM TOSN.
Mobile Sensing Structural Health Monitoring Vibration Analysis Smartphones
DeepFakeDetect
Deepfake audio detection using mechanical modeling of vocal folds

Problem: Deepfake speech can sound natural, and standard acoustic features may miss subtle production differences.

Outcome: DeepFakeDetect uses vocal-fold mechanical parameters to support interpretable audio authenticity analysis.

Venue / Status: Under review, Journal of Information Security and Applications.
Deepfake Audio Speech Security Vocal Fold Modeling Machine Learning
