Dissertation and NLP Projects

ShiftAware-Select
Reliable NLP classification under distribution shift

Text classifiers often look reliable when training and test data come from the same distribution. Real deployment is different. The model may receive unfamiliar, noisy, or shifted text and still produce a confident prediction.

ShiftAware-Select addresses this problem by combining calibration, uncertainty estimation, and selective prediction. The model can answer when confidence is supported. It can also defer uncertain cases for human review.

The main contribution is a reliability-focused decision process. It reduces the risk of confident errors under out-of-domain and noisy conditions.

Venue / Status: Under review IEEE ICMLA 2026.
Distribution Shift Calibration Selective Prediction Reliable NLP
CAGLSAv2
Discourse-span attention for length-robust sentiment analysis

Sentiment can change across a sentence or document. A later clause can reverse an earlier positive statement. Long text can also make the important sentiment evidence harder to locate.

CAGLSAv2 models discourse-span information instead of relying only on a flat sentence representation. The method is designed to capture sentiment evidence across variable-length text.

The project improves the way models handle contrast, span-level evidence, and length variation in sentiment analysis.

Venue / Status: Under review IEEE ICMLA 2026.
Sentiment Analysis Discourse Span Contrastive Clauses Context-Aware Learning
Irony-Aware Twitter Sentiment Analysis
Multi-task learning for non-literal social media text

Social media text often expresses opinion indirectly. A post may use positive words while the intended meaning is negative. Irony and emotion make this problem harder.

This project learns sentiment together with irony and emotion-related information. The model receives additional supervision about how opinion is expressed, not only which sentiment label is assigned.

The contribution is a sentiment learning framework that is more aware of non-literal language in Twitter-style text.

Venue / Status: Submission in progress, IEEE AIBThings.
Irony Emotion Multi-Task Learning Social Media NLP
DualGraph-RoBERTa
Syntactic and semantic graph fusion for aspect-based sentiment analysis

Aspect-based sentiment analysis requires a model to connect each opinion to the correct target. One sentence can praise one aspect and criticize another.

DualGraph-RoBERTa combines transformer representations with syntactic and semantic graph evidence. The model uses graph structure to keep aspect-specific opinion relations visible.

The project supports fine-grained sentiment analysis where the model must identify the target, the opinion, and the relationship between them.

Venue / Status: Submission in progress, IEEE AIBThings.
ABSA Graph Learning RoBERTa Opinion Structure
Discourse-Aware Domain Adaptation
Sentiment transfer across domains

Sentiment models often lose reliability when moved to a new domain. A restaurant review, product review, and social media post may express opinion in different ways.

This project studies transfer through discourse-aware and span-level sentiment evidence. The aim is to keep the reasoning behind a prediction visible while helping the model adapt to a new domain.

The project is part of my broader dissertation direction on reliable sentiment analysis under linguistic and domain variation.

Venue / Status: In progress.
Domain Adaptation Discourse Evidence Span-Level Sentiment Generalization

Security, Speech, and Sensing Projects

Hierarchical Severity Classification with CWE Enrichment
Evidence-aware classification for vulnerability disclosures

Vulnerability reports are technical and dense. The key evidence for severity may appear in only a few sentences, while the rest of the report gives background.

This project uses hierarchical modeling to keep sentence-level evidence visible. It also adds CWE-based security knowledge so the model can use structured vulnerability information.

The contribution is a severity classification approach that is more connected to technical evidence and more useful for security triage.

Venue / Status: Submission in progress, IEEE AIBThings.
Software Security CWE Severity Classification Sentence Evidence
LayerLens
Task-relevant layer disruption analysis for self-supervised speech encoders

Self-supervised speech encoders are widely used, but it is difficult to understand how acoustic distortion affects their internal representations. A global audio quality score does not explain which layers are disrupted.

LayerLens compares clean and distorted speech representations across layers. It uses task-relevant layer weighting to connect representation disruption with downstream task behavior.

The project provides a diagnostic view of speech models under distortion. It helps explain where and how speech encoders are affected by noise, reverberation, and enhancement artifacts.

Venue / Status: Under review IEEE/ACM TASLP.
Speech Processing Self-Supervised Learning Layer Analysis Model Diagnostics
Vi-Detect
Smartphone-based vibration sensing for component looseness detection

Detecting looseness in structural components usually requires dedicated sensing equipment. This can limit scalability and make fine-grained inspection difficult.

Vi-Detect uses smartphone accelerometers and vibration-based modeling to estimate component looseness. The method connects mobile sensing with physical behavior in structural systems.

The project shows how everyday devices can support low-cost structural health monitoring.

Venue / Status: Published on ACM SenSys 2024 poster and currently under review on ACM TOSN journal.
Mobile Sensing Structural Health Monitoring Vibration Analysis Smartphones
DeepFakeDetect
Deepfake audio detection using mechanical modeling of vocal folds

Deepfake audio detection is difficult because generated speech can sound natural to human listeners. Standard acoustic features may miss subtle physical differences in how speech is produced.

DeepFakeDetect models speech through vocal-fold mechanical parameters. The method extracts physically motivated features and uses them for fine-grained audio authenticity analysis.

The project connects speech security with interpretable modeling of the speech production process.

Venue / Status: Under review on Jorunal of Information Security and Applications.
Deepfake Audio Speech Security Vocal Fold Modeling Machine Learning
