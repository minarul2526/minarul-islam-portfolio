Research Focus

My current research investigates how machine learning systems can remain effective when facing domain shift, linguistic complexity, noise, ambiguity, and real-world uncertainty.

  • Reliable NLP: selective prediction, uncertainty estimation, and calibration
  • Context-aware sentiment analysis: contrast, irony, discourse, and hierarchical evidence
  • Speech processing: layer disruption analysis for self-supervised speech encoders
  • Security: software security, AI security, and robust decision-making
  • Sensor systems: smartphone-based structural health monitoring
  • Audio forensics: deepfake audio detection and user identification

Featured Research Projects

ShiftAware-Select

A reliable NLP classification framework under distribution shift using abstain-to-human decisioning, calibration, and uncertainty-aware selective prediction.

Focus: Reliable NLP, uncertainty estimation, selective prediction

CAGLSAv2

Context-aware learning for sentiment analysis using contrastive clause information and improved modeling of linguistic structure.

Focus: Context-aware sentiment analysis

Irony-Aware Multitask Sentiment Learning

A multitask learning framework that improves sentiment prediction by jointly modeling irony and emotion-aware signals in social media text.

Focus: Irony, emotion, and sentiment

Hierarchical Severity Classification with CWE Enrichment

A framework for software vulnerability severity prediction using hierarchical sentence evidence and structured security knowledge.

Focus: Software security and hierarchical modeling

DualGraph-RoBERTa for ABSA

A structurally informed aspect-based sentiment analysis approach that combines graph modeling with transformer representations.

Focus: Aspect-based sentiment analysis

LayerLens

A diagnostic framework for self-supervised speech encoders that measures how acoustic distortion disrupts layer-wise task-relevant representations.

Focus: Speech model diagnostics and robustness

Vi-Detect

A smartphone-based system for fine-grained vibration-based component looseness detection for structural health monitoring.

Focus: Sensing systems and structural monitoring

DeepFakeDetect

A deepfake audio detection method based on mechanical modeling of vocal folds combined with machine learning features.

Focus: Audio forensics and deepfake detection
