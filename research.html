Research Overview

My research studies how machine learning models can make more reliable decisions when language is complex, noisy, or different from the data used during training. Many current NLP models perform well on standard benchmarks, but they can still fail when text contains contrast, irony, hidden sentiment, aspect-specific opinion, or domain shift. These failures are important because they can lead to confident but wrong predictions in real-world applications.

To address this problem, my work develops context-aware and reliability-focused learning methods for sentiment analysis, text classification, and software security. The goal is not only to improve prediction accuracy, but also to make model behavior easier to understand across difficult linguistic and security settings.

Current Research Projects

ShiftAware-Select

Reliable NLP classification under distribution shift

Standard text classifiers often become overconfident when test data comes from a different distribution than the training data. This project addresses that problem by combining uncertainty estimation, calibration, and selective prediction. Instead of forcing the model to answer every input, the method allows the model to abstain on uncertain cases and defer them for human review. This improves reliability under out-of-domain and noisy conditions and reduces high-confidence errors.

CAGLSAv2

Context-aware sentiment analysis with contrastive clauses

Sentiment in real text is often shaped by contrast, such as when one clause reverses or weakens the meaning of another. A sentence-level model can miss this structure and assign the wrong sentiment. This project models contrastive clause information so the model can better understand how different parts of a sentence contribute to the final opinion. The benefit is more reliable sentiment prediction in linguistically complex text.

Irony-Aware Multitask Sentiment Learning

Sentiment prediction with irony and emotion awareness

Social media text often expresses opinion indirectly. A positive word may carry a negative meaning when the post is ironic, sarcastic, or emotionally charged. This project uses multitask learning to connect sentiment prediction with related signals such as irony and emotion. By learning these related tasks together, the model can better handle cases where surface-level words do not match the intended meaning.

Hierarchical Severity Classification with CWE Enrichment

Software vulnerability severity prediction

Security reports contain technical descriptions where the most important evidence may appear in only a few sentences. Flat text classification can hide this evidence. This project uses hierarchical modeling to capture sentence-level information and enriches the representation with CWE-based security knowledge. The goal is to support more accurate and interpretable severity classification for software vulnerabilities.

DualGraph-RoBERTa for Aspect-Based Sentiment Analysis

Structurally informed aspect-level opinion modeling

In aspect-based sentiment analysis, the same sentence may contain different opinions about different targets. A broad sentence representation can mix these opinions together. This project combines transformer representations with graph-based structure so the model can focus on the words and relations most relevant to each aspect. The method is designed to improve fine-grained opinion understanding when sentiment depends on local context and dependency structure.

Discourse-Aware Domain Adaptation for Sentiment Analysis

Generalizing sentiment models across domains

Sentiment models trained in one domain often lose accuracy when applied to another domain because opinion expressions, aspect terms, and writing styles change. This project studies domain adaptation through discourse-aware and span-level sentiment evidence. The goal is to help sentiment models transfer more reliably across domains while preserving the contextual evidence behind each prediction.
